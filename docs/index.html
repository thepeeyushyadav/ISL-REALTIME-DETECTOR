THE CUSTOM DATASET ADVANTAGE

Ground-Up Curation

Unlike generic models relying on standard Kaggle datasets (like INCLUDE), this AI was trained on thousands of frames recorded entirely from scratch by the author. This guarantees zero domain-gap for the targeted gestures.

Spatial-Temporal Augmentation

To achieve 99% accuracy in diverse environments, the dataset was artificially expanded by 500% using complex coordinate scaling, temporal shifting, and Gaussian noise injections.

DEEP LEARNING PIPELINE

Frame Capture

OpenCV extracts 30 continuous frames per sequence.

Holistic Extraction

MediaPipe isolates 225 spatial (X,Y,Z) coordinates.

LSTM Layers

A deep sequential neural network processes the temporal data.

Softmax Output

Translates sequence into English text instantly.
